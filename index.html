📋 抓取金鑰

請長按下方文字 → 全選 → 拷貝

電腦：將「抓取金鑰」拖曳至書籤列

手機：請點「複製書籤網址」，再依下方 iPhone／Android 說明手動新增書籤

▶ 使用說明
  1. 前往 星夢郵輪官網 並登入帳號
  2. 依裝置新增書籤小工具（見下方電腦／手機說明）
  3. 在官網頁面點擊書籤，金鑰會自動複製到剪貼簿
  4. 回到本頁面，貼上金鑰後點擊「認證」
  5. 選擇出發港口、人數與出發星期，點擊「開始查詢」
💻 電腦（Chrome／Edge／Safari）
  1. 將上方「抓取金鑰」按鈕拖曳至瀏覽器書籤列
📱 iPhone／iPad（Safari）
  1. 點擊「複製書籤網址」，在出現的文字框 長按 → 全選 → 拷貝
  2. 用 Safari 開啟任意網頁，點下方 分享 → 加入書籤
  3. 點底部 書籤 圖示 → 點右下角 編輯
  4. 點剛建立的書籤 → 將 網址 欄位內容全選刪除，貼上剛複製的書籤網址
  5. 名稱可改為「Sync」等，點完成儲存
  6. 之後在官網登入狀態下，點此書籤即可複製金鑰

iOS 不支援拖曳加入書籤，必須用「編輯書籤網址」方式設定。

🤖 Android（Chrome）
  1. 點擊「複製書籤網址」，在出現的文字框 長按 → 全選 → 複製
  2. 用 Chrome 開啟任意網頁，點右上角 ⋮ → 書籤 → 書籤（或先點網址列旁的 ⭐ 加入書籤）
  3. 在書籤列表找到剛加入的項目，長按 → 編輯（或點 ⋮ → 編輯）
  4. 將 網址 欄位內容全選刪除，貼上剛複製的書籤網址
  5. 名稱可改為「Sync」等，儲存
  6. 之後在官網登入狀態下，點此書籤即可複製金鑰

Android 同樣不支援拖曳書籤列；若 Chrome 無法執行書籤，可改用 Firefox，步驟類似。
